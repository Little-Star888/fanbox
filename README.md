# 📦 翻箱 FanBox

> vibe coding 的本地文件指挥中心。在浏览器里查看、搜索、预览、一键打开本机所有文件夹和文件。

AI 帮你一个下午起十个项目，但它们散在各处、名字认不出、打开还得右键好几步。翻箱补上这块：**快速找回、原地验货、一键抵达**。

## 快速开始

### 最简单：双击启动（不用碰命令行）

- **macOS**：双击 `启动翻箱.command`
- **Windows**：双击 `启动翻箱.bat`

浏览器会自动打开 `http://localhost:4567`，完事。用完关掉弹出的小黑窗口就退出了。

> **还没装 Node.js？** 双击启动时如果提示没装，去 [nodejs.org](https://nodejs.org) 下载 LTS 版本，一路点「继续 / 下一步」装好，再回来双击一次就行。它是运行翻箱的免费基础环境，装一次永久有效。
> （macOS 首次双击若被安全拦截：右键 → 打开 → 确认打开。）

### 命令行方式

```bash
node server.js
```

> 换端口：`FANBOX_PORT=8080 node server.js`　·　不自动开浏览器：`FANBOX_NO_OPEN=1 node server.js`

## 核心功能

- **⌘K 全局模糊搜索** — 文件和**文件夹**都能找回，记得名字片段就行；`⌘↵` 直接用编辑器整包打开找到的项目；输入 `内容:关键词` 切换全文内容搜索。
- **原地预览** — Markdown 渲染、HTML 实时预览成品、代码语法高亮、图片/视频/PDF 内嵌。
- **一键打开** — 默认应用 / VS Code 编辑器 / 在 Finder 显示 / 复制路径。
- **在此打开终端** — 顶栏一键在当前目录唤起系统终端，找回项目后直接去跑。
- **网格 + 列表双视图** — 图片显缩略图；可按名称/时间/大小排序。
- **项目识别** — 自动给 Node / Python / Web / Rust / Go 项目打徽章。
- **收藏 + 最近** — 常用目录收藏常驻；「最近修改」还原你刚做了什么。
- **三套皮肤一键切换**（侧栏左下角）——配色、字体、图标、代码高亮整体随之变化，选择记忆在本地：
  - **终端** · Volt 荧光绿 × 炭黑 × 等宽字，工业仪器面板感（默认）
  - **档案** · 奶油纸 × 赤陶橙 × 衬线，温暖纸感档案馆（参照 Anthropic 暖色出版）
  - **索引** · 黑白 × 信号红/绿 × 巨号字，编辑式索引日报（参照 Bloomberg Businessweek）

## 快捷键

| 操作 | 键 |
|---|---|
| 全局搜索 | `⌘K` |
| 当前目录筛选 | `/` |
| 结果上下选择 | `↑` `↓` |
| 打开/预览 | `↵` |
| 用编辑器打开 | `⌘↵` |
| 后退 | `⌘[` |
| 关闭 | `Esc` |

## 隐私与安全

- 只在本机回环地址 `127.0.0.1` 监听，**数据不出本机**，不上传任何东西。
- 收藏与最近记录存在 `~/.fanbox/config.json`。
- 默认不做任何文件增删改，只读 + 调用系统「打开」，避免误操作。

## 设计取舍

翻箱不跟 Finder 拼文件操作（剪切/重命名/批量移动），专注「查看 + 找回 + 打开」这一条链路做到顺手。不做云、不做远程、不做账号——本地、零配置、零依赖。
